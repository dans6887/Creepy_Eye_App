# Creepy_Eye_App
A small app that displays a creepy eye when the user clicks a button.

Using PYQT6 I creeted a small application that displays a creepy, searching eye on your screen. 
The user can turn the eye on and off with the click of a button.

This is one of my first gui-based python appplications.
